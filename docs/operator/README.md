## Current Operating Position

DCA OS Lite is ready for controlled local/admin MVP work.

It is not automatically approved for production deployment or live client portal rollout.

## Admin Rule

The admin controls the process. AI helps prepare research, plans, drafts, images, summaries, and handoff material, but the admin reviews and decides what becomes final.

## Client Rule

Clients should see final approved work only.

The current Client Portal is read-only and should not be treated as an approval/comment system.

## Production Rule

Production, VPS, live client access, live provider sending, and live external integrations need a separate approval block before use.
