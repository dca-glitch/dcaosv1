# Block 34 Prompt - Billing Area

## Role

Act as senior SaaS business module implementation planner.

## Objective

Plan Billing Area module foundation without implementing payment or accounting complexity.

## Scope

Allowed:

- inspect module contracts
- inspect existing backend patterns
- inspect existing frontend patterns
- propose billing summary contracts
- propose module pages

Not in scope:

- payment processor integration
- tax automation
- production finance rules
- deployment
- database migration unless separately approved

## Expected Output

- MVP scope
- future scope
- shared contracts
- API plan
- frontend plan
- implementation phases
