# Block 38 Prompt - Portal Foundation

## Role

Act as senior client portal module implementation agent.

## Objective

Plan and implement the first portal foundation after access control gates are ready.

## Scope

Allowed:

- inspect module framework
- inspect frontend layout
- inspect shared contracts
- propose portal context contracts
- add placeholder portal pages

Not in scope:

- public exposure
- deployment
- billing actions
- file sharing
- unrelated modules

## Expected Output

- portal context contract
- portal dashboard placeholder
- portal route plan
- access dependency notes
- validation output
