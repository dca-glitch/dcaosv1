## Practice run steps

1. Pick one test client and one test month.
2. Create monthly AI Delivery project + brief.
3. Prepare content plan → draft → final deliverable.
4. Prepare monthly report summary; finalize when client-safe.
5. Verify Client Portal archive view.
6. Record confusion log in operator notes (not committed secrets).

## Done when

Admin can explain: what was created, where final work lives, what the client sees, what still needs improvement.

## Stop conditions

Stop if client, month, project, or final item becomes unclear. No deploy/VPS.
