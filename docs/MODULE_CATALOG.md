## Current State Note

AI Delivery is admin/operator-primary and local-first. **MVP 1 = Puriva client delivery** (`puriva.id`). **Client Portal MVP is required for Puriva** — client-safe visibility and human/client review before publication (Market Intelligence summary, AI SEO status, Google Docs deliverables, website publishing handoff/status, product catalog inquiry). Advanced portal features (magic links, full comment threads) remain phased. Future modules (Spa Finance, BaliShop, GayService, GotoBeauty, Bali24, Revenue Hub, Commerce Core) are strategic/module-map items, not current implementation scope. No live AI calls, crawling, WordPress auto-publish, GA/GSC, Resend sending, or production deployment is active unless explicitly approved.
