# Botanical Light Product UI Direction

**Status:** current UI authority for DCA OS Lite.

## Current truth

Botanical Light is the current product UI direction.

It supersedes Dark Nebula as the current visual language for canonical documentation and external-agent guidance.
Dark Nebula documents remain historical only where they preserve audit or deployment provenance.

## What Botanical Light means in this repository

- light-first product UI
- English-only interface terminology
- shared reusable UI foundation rather than per-module visual systems
- role-appropriate density: admin/operator surfaces stay efficient; client surfaces stay clear and client-safe
- responsive proof references at **1440**, **768**, and **390**
- ongoing modal-to-page migration where full-page workflows are clearer
- retained confirmation and other single-purpose overlays where a modal is still the safest interaction

## Usage rules

- If a current document conflicts with older Dark Nebula guidance, Botanical Light wins.
- If current implementation evidence conflicts with older mockups or audit packs, the proven current implementation wins.
- New frontend work should extend shared primitives/components instead of inventing a separate visual system.
